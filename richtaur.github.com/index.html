Matt Hackett's little playground on GitHub

Hi I'm Matt Hackett and this is my GitHub page where I'll be putting up mostly game experiments and prototypes. More about me can be found on my personal site.

Demos

  • QBASIC: a QBASIC-esque library for JavaScript.
  • DRAGON: an old QBASIC game I wrote in 1996, partially translated into HTML5 (using the above library).
  • Spacius!: A Space Adventure. Get the soundtrack!
  • Turn-Based Canvas Demo: an experiment with some turn-based strategy gameplay.

Play Bombada

Bombada is an open source match-3 game I made. I designed it, wrote the JavaScript and made the graphics. Music and sound effects by Joshua Morse.
